## 수행 업무
* 게시글 상세 보기 페이지 개발
  - 일반 판매 게시글과 경매 게시글의 페이지를 구분해서 작성했습니다.
  - mybatis 라이브러리를 사용해 DB에 저장되어 있는 게시글 정보, 연관된 사진 데이터 가져와 페이지에 표시되도록 하였습니다.
  - 신고, 찜하기 기능들을 Ajax를 통한 비동기로 처리되도록 하였습니다.
* 쪽지 시스템 개발 (대화방 형식, 실시간 화면 반영 X)
  - 상세 보기에서 쪽지 보내기로 이동시에 특정 게시물과 인원들이 연락 중인 쪽지 리스트 출력합니다.
  - 게시물과 대화 상대방을 구분하여 연락 리스트 및 마지막 대화내용 출력합니다.
  - 본인에게만 보여지지 않도록 쪽지 삭제를 구현했습니다.
  - 사진 전송 기능을 추가했습니다.
  - 신고 기능을 추가했습니다.
  - 거래 처리 시스템 동작 처리 폼 추가했습니다.
* 경매 시스템 개발
  - 경매 게시글 상세 페이지에서 경매 시스템 동작합니다.
  - 실시간으로 처리
  - 
* 거래 처리 시스템 개발
* 페이 관리 페이지 개발
* 관리자 페이지 디자인
* 쪽지 관리 페이지 개발
* 신고 관리 페이지 개발
* 전체적인 히스토리 검색 페이지 개발
